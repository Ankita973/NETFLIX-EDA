# 🎬 Netflix Exploratory Data Analysis (EDA)

This project performs an in-depth **Exploratory Data Analysis (EDA)** on the Netflix Titles dataset to uncover trends in content production, countries, genres, directors, and release patterns using Python.

---

## 📌 Project Objective

The objective of this project is to analyze Netflix's content library and answer business-oriented questions through data cleaning, visualization, and statistical exploration.

---

## 📂 Dataset

- Dataset: Netflix Titles Dataset
- Format: CSV
- Records: Movies and TV Shows available on Netflix
- Features include:
  - Title
  - Type
  - Director
  - Cast
  - Country
  - Release Year
  - Date Added
  - Rating
  - Duration
  - Genre

---

## 🛠️ Technologies Used

- Python
- NumPy

- Matplotlib

- Jupyter Notebook

---

## 📊 Data Preprocessing

The following preprocessing steps were performed:

- Imported and explored the dataset
- Checked dataset dimensions and data types
- Identified missing values
- Removed duplicates (if any)
- Converted date columns into datetime format
- Cleaned categorical columns
- Prepared data for visualization

---

## 📈 Exploratory Data Analysis

The project answers several important analytical questions, including:

### Content Analysis
- Movies vs TV Shows distribution
- Growth of Netflix content over the years
- Year-wise content additions
- Content release trends

### Country Analysis
- Countries producing Netflix content
- Top countries with the highest number of Movies
- Top countries with the highest number of TV Shows

### Genre Analysis
- Most popular genres
- Genre distribution

### Director Analysis
- Total number of directors
- Directors with the highest number of titles

### Duration Analysis
- Most common movie duration
- Distribution of TV Show seasons

### Time Analysis
- Gap between content release date and Netflix addition date
- Oldest and newest content available

## 📌 Key Insights

- Movies make up the majority of Netflix's content library.
- Content production has increased significantly since 2015.
- The United States contributes the largest share of Netflix titles.
- Drama and International Movies are among the most popular genres.
- Most movies have a runtime between 80–120 minutes.
- Many TV Shows consist of only one season.
